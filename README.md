# Superstore Sales Dashboard

## Overview
This project is a Power BI dashboard built using the Superstore dataset.

The dashboard provides insights into:
- Sales performance
- Profitability
- Regional trends
- Category analysis
- Year-over-Year growth

---

## Data Model
A star schema was created using:
- Fact_Orders
- Dim_Date
- Dim_Region
- Dim_Category
- Dim_Segment

---

## DAX Measures
Key measures used:
- Total Sales
- Total Profit
- Profit Margin %
- Sales LY
- Sales YoY %

---

## Dashboard Features
- KPI Cards
- Sales Trend Analysis
- Regional Performance
- Segment Breakdown
- Interactive Slicers
